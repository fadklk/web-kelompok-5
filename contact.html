/ ABOUT ME

1.IT
Information Technology merupakan sebuah teknik untuk menyiapkan, memproses, mengumpulkan, menganalisis, dan menyebarkan informasi dengan teknologi.

2.RAM
Random Access Memory merupakan hardware yang ada di dalam perangkat. RAM berfungsi untuk menyimpan data sementara yang bekerja saat perangkat tersebut beroperasi.

3.ROM
Read Only Memori adalah perangkat keras yang digunakan untuk media penyimpanan pada komputer yang bersifat permanen.

4.GUI
Graphical User Interface, yakni tampilan untuk program berbasis grafis

5.IC
Rangkaian mikroelektronik yang berupa semikonduktor yang terdiri dari sejumlah besar transistor dan komponen lainnya

6.DLNA
Digital Living Network Alliance, fitur yang digunakan untuk mengakses dokumen multimedia seperti musik, video, film dan gambar yang ada di perangkat lain dengan koneksi Wifi atau router.

7.ISDN
Integrated Services Digital Network, cara untuk transfer data dengan kecepatan lebih tinggi melalui saluran telepon dengan kecepatan transfer 128.000bps (bit per detik).

8.PPI
Pixel per inch, jumlah pixel di layar sekaligus yang jadi satuan kepadatan pixel dalam sebuah gambar digital.

9.HSPDA
High-Speed Downlink Packet Access adalah sebuah protokol telepon genggam dan kadang kala disebut sebagai teknologi 3,5G. HSDPA adalah evolusi dari W-CDMA. Ini dirancang untuk meningkatkan kecepatan transfer data 5x lebih tinggi.

10.UMTS
Universal Mobile Telecommunications System, salah satu teknologi telepon genggam 3G

11.DNS
Sebuah layanan nama domain alamat IP untuk nama domain web.

12.LAN
Local Area Network atau jaringan area lokal yang dapat menghubungkan perangkat ke jaringan bersama untuk berbagi data di dalam jaringan.

13.FIREWALL
Sistem keamanan yang melindungi akses yang kurang sah pada jaringanmu.

14.API
Antarmuka Pemrograman Aplikasi, memungkinkan fitur perangkat lunak diakses oleh aplikasi dan bekerja sesuai kebutuhan.

15.COOKIE
kumpulan informasi yang berisi history dan aktivitas ketika menelusuri sebuah website

16.BANDWIDTH
Kecepatan transfer data maksimum dari jaringan internet.

17.VPN
Virtual Private Network atau VPN, layanan yang memungkinkan pengguna untuk akses situs secara pribadi melalui server jaringan lain.

18.CLOUD HOSTING
Web hosting yang digunakan untuk resource dari beberapa server untuk menyeimbangkan beban, mempercepat performa, dan memaksimalkan uptime.

19.ACCESS POINT
Perangkat keras yang digunakan dalam jaringan area lokal nirkabel untuk mengirim dan menerima data

20.SERVER HOSTING
Tempat untuk menyimpan data pada website dan ditempatkan dalam sebuah server
